# The Waroenkz - E-Commerce Application

This is an E-Commerce web application built with React and Vite. It provides a complete shopping experience with features for different types of users including Guests, Members, and Administrators.

## Features

- **User Authentication**: Login and Registration system.
- **Product Catalog**: Browse products with category filters.
- **Shopping Cart**: Add products to cart and manage quantities.
- **User Roles**: 
  - Admin: Manage products and categories.
  - Member: View purchase history and shop.
  - Guest: Browse products.

## Tech Stack

- **Frontend**: React, Vite
- **Styling**: Tailwind CSS
- **State Management**: React Context API

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
