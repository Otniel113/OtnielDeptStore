# OtnielDeptStore
Otniel Department Store. Beberapa toko yang dijadikan dalam satu lingkup department store.

## Deployment
Menggunakan Github Pages pada link: https://otnielabiezer.com/OtnielDeptStore/

## Toko-toko
1. The Waroenkz = Aplikasi kasir untuk checkout dan bisa mengetahui sisa persediaan
2. Tokotniel = Aplikasi untuk topup saldo dan melakukan pembelian online
